Python Cheatsheet

Die Syntax für eure Projekte

1. Variablen

Zum Speichern von einzelnen Werten (Zahlen, Text).

# Text speichern name = "Mario" # Zahl speichern punkte = 0 # Rechnen punkte = punkte + 1

2. Sequenz (Ein/Ausgabe)

Befehle werden nacheinander ausgeführt.

print("Hallo!") name = input("Wie heißt du? ") print("Hallo " + name)

3. Verzweigung (If/Else)

Entscheidungen treffen.

alter = 16 if alter >= 18: print("Du bist volljährig.") else: print("Du bist noch minderjährig.")

4. Schleife (Loop)

Dinge wiederholen.

# Zählschleife (z.B. 5 mal) for i in range(5): print("Hallo") # Bedingte Schleife (Solange...) running = True while running: befehl = input("Ende? (j/n)") if befehl == "j": running = False

5. Listen

Viele Werte in einer Variable speichern.

# Liste erstellen freunde = ["Anna", "Tom", "Lisa"] # Etwas hinzufügen freunde.append("Kevin") # Alles anzeigen (mit Schleife) for freund in freunde: print(freund)
